Tyler's TicTacToe

Rules

Standard tic-tac-toe game rules. Play to get three in a row vertically, horizontally, or diagonally.

You cannont place your piece where a piece already is and you cannot redo a move.

About This Game

This game was created on the Node.js platform using Express.js to run the server. The game is coded using plain old Javascript and the front end was designed using Bootstrap 4. This web app was developed and is maintained by Tyler Ingham, and is hosted using Heroku. Please feel free to check out the links below to see the technology used. Thank you and have a great time playing classic Tic Tac Toe!
